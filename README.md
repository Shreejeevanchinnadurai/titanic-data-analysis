# Titanic Data Analysis

## Tools Used
- Python
- Pandas
- NumPy

## What I did
- Loaded and explored Titanic dataset
- Cleaned missing values (Age, Embarked, Cabin)
- Analyzed survival rates by Gender, Class, Age
- Found key insights using GroupBy

## Key Findings
1. Women had 74% survival rate vs men 19%
2. Class 1 passengers survived most (63%)
3. Children had highest survival rate (54%)
4. Port C passengers had highest survival (55%)
5. Class 3 had most passengers (491)
